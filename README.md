# NaiyeBharat — Legal Advocacy Platform

A premium legal services platform built with **Next.js**, offering comprehensive legal consultation, counselling, and case management services.

## Features
- Expert legal advocacy & case management
- Mental health counselling integration  
- Court marriage registration assistance
- Emergency legal response
- Secure client portal

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: GSAP ScrollTrigger
- **Runtime**: Node.js 20

## Local Development
```bash
cd webapp
npm install
npm run dev
```
